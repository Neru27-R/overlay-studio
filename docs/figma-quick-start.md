# Figma Quick Start

1. Start the local preview with `start-preview.cmd`.
2. Use the client URL for public-facing design work:

```text
http://127.0.0.1:5173
```

3. Capture or import the client page into Figma with your preferred HTML/import plugin.
4. Keep the admin URL private. Do not place it in public docs, shared Figma notes, or screenshots meant for clients.

Most visual styling lives in:

```text
src/styles/global.css
```
